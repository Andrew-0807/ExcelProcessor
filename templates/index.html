How to Use

Upload Files

Drag and drop your Excel files (.xlsx, .xls, .csv) into the upload area, or click to browse and select files from your computer.

Select Processing Mode

Choose the appropriate processing mode for your needs:

  • Adaos: Add percentage markup to values
  • SGR: Apply SGR processing rules
  • Minus: Subtract specified values
  • Extract: Extract specific data
  • Borderou: Transform borderou format
  • CardCec: Process card data

Process Files

Click the "Process Files" button to start processing. The processed files will be available for download once complete.
